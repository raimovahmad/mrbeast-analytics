# 🎬 MrBeast YouTube Analytics Dashboard

Live dashboard analyzing MrBeast's YouTube channel performance (44 videos, 11.75B views).

## 🔗 Live Demo
👉 https://raimovahmad.github.io/mrbeast-analytics/

## 📊 What's Inside
- Views, likes, comments analysis
- Keyword impact on views ("world" → 344M avg)
- Shorts vs Long vs Medium format comparison
- Top 10 videos leaderboard
- Analyst insights with real data

## 🛠 Tech Stack
- Python (google-api-python-client, pandas, scikit-learn)
- YouTube Data API v3
- Chart.js

## 💡 Key Findings
- Shorts average 341M views — 40% more than Long-form
- "world" keyword = highest avg views (344M)
- Emotional content drives highest like rate (3.89%)
